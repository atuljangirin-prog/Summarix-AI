# 📄 Summarix AI — AI PDF Summarizer

Summarix AI is a modern AI-powered PDF Summarizer web application built using **HTML5, CSS3, and Vanilla JavaScript** with a clean SaaS-style UI, smooth performance, real PDF text extraction, intelligent summarization logic, and responsive dashboard experience.

Summarix AI:

2. Processes document content
3. Detects important sentences
4. Identifies keywords/topics
5. Generates readable summaries
6. Displays insights and analytics
